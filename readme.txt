# Описание задачи

Программа на языке C++, которая обрабатывает входные данные из текстового файла, описывающего работу компьютерного клуба. Программа должна вывести список событий, произошедших за рабочий день, выручку за день и время, которое каждый стол был занят.

## Входные данные

Формат входного файла следующий:
1. Количество столов в компьютерном клубе.
2. Время начала и окончания работы компьютерного клуба.
3. Стоимость часа в компьютерном клубе.
4. Список входящих событий: время события, идентификатор события, тело события.

Подробный формат входных данных описан в задании.

## Выходные данные

Если входные данные корректны, программа выводит:
1. Время начала работы.
2. Все события за рабочий день.
3. Время окончания работы.
4. Информацию о каждом столе: номер стола, выручка за день, время использования.

## Инструкция по сборке и запуску

Для сборки программы используется предоставленный makefile. 

1. Для компиляции программы, выполните команду make.
2. Для запуска программы с тестовым файлом test_file.txt, выполните команду make test.
3. Для удаления предыдущего исполняемого файла, выполните команду make clean.
